# User Auth Backend

This is a **Node.js + Express** backend for user authentication and profile management, built with **MongoDB Atlas**. It supports JWT-based authentication and CRUD operations for user profiles and preferences.

---

## Features

- User registration with password hashing  
- User login with JWT authentication  
- Fetch logged-in user profile (protected route)  
- Update user profile and preferences  
- Delete user account  
- MongoDB Atlas connection

---

## Tech Stack

- Node.js  
- Express.js  
- MongoDB Atlas  
- Mongoose  
- bcryptjs (for password hashing)  
- JSON Web Tokens (JWT)  
- express-validator
